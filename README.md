# CapMetro Data Sonification Project

## by Surain Saigal

This project aims to programatically sonify the CapMetro bus transit system in Austin, Texas. This repo includes the code used to collect and process this data, available through the Texas Open Data Portal. This repo also includes the code used to generate audio and visuals that illustrate the data.

The [web page](https://surainsaigal.github.io/capmetro-sonification) for the final project gallery is also hosted here, please take a look if you like!

This project was completed as a final requirement and Connecting Experience for my Digital Arts and Media certificate at UT Austin.
